# 🎯 AI-ERP 사용 가이드

## 📌 프롬프트 입력의 스마트 동작

AI-ERP의 프롬프트는 **상황에 따라 자동으로 다르게 동작**합니다!

### 1️⃣ 메뉴가 없을 때 (첫 시작)

**동작**: 새 화면 생성

**예시:**
```
사용자 등록 화면 만들어줘
제품 재고 관리 화면 만들어줘
주문 관리 화면 만들어줘
```

**결과**: 
- ✅ AI가 자동으로 필요한 필드 생성
- ✅ 그리드 화면 생성
- ✅ 사이드바에 메뉴 추가

---

### 2️⃣ 메뉴를 선택한 후

**동작**: 데이터 추가 (AI가 자동으로 파싱)

**예시 1: 쉼표로 구분**
```
홍길동, hong@company.com, 010-1234-5678
김철수, kim@company.com, 010-9876-5432
```

**예시 2: 자연어로 입력**
```
홍길동 부장님 이메일은 hong@company.com이고 
전화번호는 010-1234-5678입니다.
마케팅팀 소속이며 2020년 입사했습니다.
```

**예시 3: 여러 항목 한번에**
```
다음 사용자들을 추가해줘:
- 홍길동 (hong@company.com, 010-1234-5678)
- 김철수 (kim@company.com, 010-9876-5432)
- 이영희 (lee@company.com, 010-1111-2222)
```

**결과**:
- ✅ AI가 자동으로 텍스트 파싱
- ✅ 스키마에 맞게 데이터 매칭
- ✅ 그리드에 자동 추가
- ✅ 신뢰도 표시 (70% 미만이면 확인 요청)

## 🎨 전체 워크플로우

### Step 1: 화면 생성
```
프롬프트: "사용자 등록 화면 만들어줘"
           ↓
        AI 분석
           ↓
      스키마 생성
           ↓
      화면 완성! 🎉
```

### Step 2: 데이터 입력 (3가지 방법)

#### 방법 A: 프롬프트로 입력
```
프롬프트: "홍길동, hong@company.com, 010-1234-5678"
           ↓
        AI 파싱
           ↓
      데이터 저장
           ↓
      그리드에 표시! ✅
```

#### 방법 B: 엑셀 파일 업로드
```
엑셀 파일 드래그 앤 드롭
           ↓
      파일 파싱
           ↓
    AI 컬럼 매칭
           ↓
    대량 데이터 저장
           ↓
      그리드에 표시! ✅
```

#### 방법 C: 수동 입력
```
"➕ 새 항목 추가" 클릭
           ↓
      폼 입력
           ↓
      저장 버튼
           ↓
      그리드에 표시! ✅
```

## 💡 팁과 트릭

### 1. 새 화면을 만들고 싶을 때
- 사이드바에서 빈 공간 클릭 (메뉴 선택 해제)
- 또는 다른 메뉴로 전환

### 2. 데이터 입력 시
- 쉼표로 구분하면 AI가 순서대로 매칭
- 자연어로 입력하면 AI가 의미 파악
- 필드명을 명시하면 더 정확함

### 3. 신뢰도가 낮을 때
- 필드명을 명확히 입력
- 예: "이름: 홍길동, 이메일: hong@company.com"

### 4. 여러 데이터 한번에
- 줄바꿈으로 구분
- 또는 번호 매기기 (1. 2. 3.)
- 또는 대시 사용 (- - -)

## 🎯 실전 예시

### 예시 1: 사용자 관리

**1단계: 화면 생성**
```
사용자 등록 화면 만들어줘
```

**2단계: 데이터 추가**
```
홍길동, hong@company.com, 010-1234-5678, 마케팅팀, 부장
김철수, kim@company.com, 010-9876-5432, 개발팀, 과장
이영희, lee@company.com, 010-1111-2222, 인사팀, 대리
```

**3단계: 검색 및 편집**
- 검색창에 "홍길동" 입력
- ✏️ 버튼으로 수정
- 🗑️ 버튼으로 삭제

---

### 예시 2: 제품 관리

**1단계: 화면 생성**
```
제품 재고 관리 화면 만들어줘
```

**2단계: 데이터 추가**
```
노트북 A, 1500000, 50, 전자제품
마우스 B, 25000, 200, 주변기기
키보드 C, 80000, 150, 주변기기
```

**3단계: 엑셀 업로드**
- products.xlsx 드래그 앤 드롭
- AI가 자동으로 컬럼 매칭
- 대량 데이터 일괄 저장

---

### 예시 3: 주문 관리

**1단계: 화면 생성**
```
주문 관리 화면 만들어줘
```

**2단계: 자연어로 입력**
```
홍길동 고객이 노트북 A를 2대 주문했습니다.
총 금액은 3,000,000원이고 배송지는 서울시 강남구입니다.
결제는 카드로 완료되었습니다.
```

**3단계: 상태 관리**
- 주문 상태를 "배송중"으로 변경
- 송장번호 입력
- 배송 완료 처리

## 🚀 고급 기능

### 1. 관계형 데이터 (향후 지원)
```
주문 화면에서 고객 정보 연결
제품 화면에서 카테고리 연결
```

### 2. 워크플로우 자동화 (향후 지원)
```
주문 생성 시 자동으로 이메일 발송
재고 부족 시 자동으로 발주 생성
```

### 3. 대시보드 (향후 지원)
```
매출 통계 차트
재고 현황 그래프
주문 추이 분석
```

## ❓ 자주 묻는 질문

### Q: 프롬프트에 뭘 입력해야 할지 모르겠어요
**A**: 상황에 따라 다릅니다!
- 메뉴 없을 때: "○○ 화면 만들어줘"
- 메뉴 선택 후: 데이터를 자유롭게 입력

### Q: 데이터가 잘못 저장되었어요
**A**: 
- ✏️ 버튼으로 수정 가능
- 🗑️ 버튼으로 삭제 후 재입력

### Q: AI가 필드를 잘못 매칭했어요
**A**: 
- 필드명을 명확히 입력: "이름: 홍길동"
- 또는 수동으로 "➕ 새 항목 추가"

### Q: 엑셀 파일이 업로드 안 돼요
**A**: 
- 먼저 화면을 생성하세요
- 메뉴를 선택한 상태에서 업로드
- .xlsx, .xls, .csv 파일만 지원
