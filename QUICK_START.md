# 🚀 빠른 시작 가이드

## 1️⃣ 환경 설정 (5분)

### API 키 발급 (무료!)

**Gemini API 키 발급** (추천 - 완전 무료!)

1. https://makersuite.google.com/app/apikey 접속
2. "Create API Key" 클릭
3. API 키 복사

### .env 파일 생성

프로젝트 루트에 `.env` 파일을 생성하고 다음 내용을 붙여넣으세요:

```env
# Server
PORT=3001
NODE_ENV=development

# AI (Gemini - 무료!)
DEFAULT_AI=gemini
GEMINI_API_KEY=여기에_발급받은_API_키를_붙여넣으세요

# Data
DATA_DIR=./data
PERSIST_INTERVAL=60000

# Frontend
VITE_API_URL=http://localhost:3001/api
```

## 2️⃣ 실행 (1분)

```bash
# 의존성 설치 (이미 완료됨)
npm install

# 개발 서버 실행
npm run dev
```

브라우저가 자동으로 열립니다!
- 프론트엔드: http://localhost:5173
- 백엔드: http://localhost:3001

## 3️⃣ 첫 화면 만들기 (30초)

1. 프롬프트 입력창에 다음을 입력:
   ```
   사용자 등록 화면 만들어줘
   ```

2. "✨ 생성" 버튼 클릭

3. AI가 자동으로:
   - 필요한 필드 생성 (이름, 이메일, 전화번호 등)
   - 그리드 화면 생성
   - 데이터 입력 준비 완료!

## 4️⃣ 데이터 입력하기

### 방법 1: 수동 입력
1. "➕ 새 항목 추가" 버튼 클릭
2. 필드 입력
3. 💾 저장

### 방법 2: 엑셀 파일 업로드
1. 엑셀 파일을 프롬프트 영역에 드래그 앤 드롭
2. AI가 자동으로 컬럼 매칭
3. 데이터 자동 저장!

### 방법 3: 텍스트로 입력
프롬프트에 다음과 같이 입력:
```
홍길동, hong@company.com, 010-1234-5678
```
AI가 자동으로 파싱하여 저장!

## ⚠️ 문제 해결

### 서버가 시작되지 않아요
- `.env` 파일이 있는지 확인
- `GEMINI_API_KEY`가 올바른지 확인
- 포트 3001이 사용 중인지 확인: `lsof -i :3001`

### AI가 응답하지 않아요
- API 키가 유효한지 확인
- 인터넷 연결 확인
- 브라우저 콘솔에서 에러 확인 (F12)

### 데이터가 저장되지 않아요
- `data/` 폴더 권한 확인
- 서버 로그 확인
